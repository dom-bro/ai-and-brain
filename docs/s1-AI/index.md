# 🤖 AI 狂飙之路

> 从一个夏天的小型研讨会，到一个让全世界为之屏息的产业。

## 本章导读

本章将沿着时间线，回顾 AI 从概念诞生到今天大模型时代的关键节点：

- 1956：达特茅斯会议与 "AI" 这个词的诞生
- 早期符号主义与连接主义之争
- 神经网络的两次寒冬与三次复兴
- 深度学习的崛起：AlexNet、AlphaGo 与 Transformer
- 大模型时代：从 GPT 到多模态、Agent 与具身智能

## 你将会读到

- 为什么 AI 会经历"寒冬"，又是什么让它每次都活了过来？
- 神经网络从生物启发，到工程胜利，中间隔着哪些关键的"想法"？
- 当一个模型有几千亿参数时，它到底学到了什么？
